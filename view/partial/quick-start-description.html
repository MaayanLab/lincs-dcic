A collection of common LINCS tools providing access to various aspects of the LINCS data and software tools.

Lists of up- and down-regulated genes can be analyzed in context of the L1000 gene expression dataset. More broadly, LINCS data can be explored using standard statistical analyses or data visualizations.

Data from multiple high-throughput assays on the perturbational effects of experimental conditions, such as particular cell lines and small-molecules, can be queried using text-based searches across multiple tools.

Once data of interest is found, it can be downloaded or analyzed in a user-friendly and web-based environment.
