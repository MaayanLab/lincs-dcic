BD2K-LINCS DCIC External Data Science Research Projects

Request for Applications

Purpose

The BD2K-LINCS Data Coordination and Integration Center (DCIC) is soliciting applications from the scientific community for data science research collaborations with the DCIC that focus on mining and integrating data generated by the LINCS program for new scientific discovery. The overarching goal of this solicitation is to maximize the value of LINCS data and transform it into new knowledge. Suitable research topics may include, but are not limited to: new approaches for data visualization and data mining; data integration and comparison with non-LINCS data; computational modeling of LINCS data.

Details about the LINCS project including the types of data currently available as well as anticipated data release timelines are available on this page. The DCIC, via a subcontract, will support collaboration projects with a budget limit of $130K in total costs per year for up to two years.

Scope of the Projects

The proposed project should clearly articulate how, if successful, it will enhance the value of the LINCS data to the community. Experimental validation of the results of the data science efforts is not required, but, can be proposed. However, funds for any such experimental work cannot be requested as part of this proposal. Information about the LINCS data sets (currently available, and in process of being generated) along with videos that describe the scientific aims from the LINCS and DCIC PIs are available at on this website.

Examples of potential projects include:

  • Processing and analysis of imaging data or integration of imaging data with other data modalities
  • Methods for integration and analysis of datasets from across regulatory layers
  • Utilization of machine learning methods that link molecular data with cell and organism phenotypic data
  • Methods that integrate LINCS data with other datasets to enable specific translational goals
  • Dynamical models of iPS cells and cellular engineering with small molecules
  • Utilization of crowdsourcing technologies to integrate and analyze large-scale molecular, imaging and phenotypic data
  • Advance web-technologies for data search and visualization
  • Collaborations on development of ontologies, metadata annotation, and data standards for data management of LINCS data
  • Integration of LINCS data with other publicly available large data resources

However, other innovative approaches for dealing with Big Data science and are related to LINCS will be considered. Please note that if your proposal is solely focused on transcriptomics data, it would have to be particularly innovative.

Point of Access to LINCS Data

To learn more about the LINCS project and the data that is collected by the LINCS Data Signature Generation Centers (DSGCs) and the activities of BD2K-LINCS DCIC, potential applicants should refer to this website. Pre-existing collaborations with the LINCS centers is not a requirement for a successful application, however, the applicant should demonstrate sufficient familiarity with the data types and other relevant experimental constraints.

Solicitation Procedures

The NIH LINCS project team and the PIs plan to advertise this opportunity in a coordinated manner with the aim of disseminating it widely to the biomedical research community.

Application Instructions

The application must be written in a 5 page document including figures and tables. The first page must contain the Specific Aims. The remaining pages are reserved for the description of the research strategy. Font must be Arial 11 or higher, margins must be set to “Narrow”, and spacing between text lines must be set to “Single” in Microsoft Word. References can be listed as additional pages.

Detailed budget for the first and second year and biographical sketches of the key investigators must be provided using NIH face pages 4 and 5 and the Biographical Sketch Format Page template as provided here.

Please submit your application by e-mail as a single PDF file on this site's application form. Mechanisms to upload application will be available at the opening date until the due date. The assembled application should be submitted grants@lincs-dcic.org. Applications that do not adhere to these guidelines will be rejected.

Review Process

A fair review process organized by the NIH Project Team will be established by recruiting impartial mail reviewers. These mail reviewers will score applications and provide brief summary statements. The final decision will be made by a formal discussion between the LINCS steering committee, NIH scientific officers, and the LINCS External Scientific Panel and will be guided by the scores and evaluation the application received.

Award Budget

Budget requests should be suitable to support funds for one postdoctoral fellow (or equivalent) including travel to one of the annual LINCS or BD2K meetings. We expect that direct costs are limited to a maximum of $100K in each year.

Award Project Period

The maximum project period is for 2 years. There will be no renewals.

Anticipated Number of Awards

The intention is to fund 4-5 awards.
